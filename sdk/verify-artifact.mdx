# verifyArtifact

verifyArtifact verifies artifact hash references, proof metadata, and evidence-linked execution records.

    const verified = await client.verifyArtifact({
      artifactHash: "sha3-512:example",
      proofHash: "sha3-512:example"
    });

This method supports independent verification, proof review, and audit support.
