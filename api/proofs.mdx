# Proof API

Proof objects may include public-safe references such as:

- Decision hash
- Artifact hash
- Execution hash
- Audit hash
- Lineage root
- Merkle root
- Timestamp
- Policy profile
- Verification status

These fields support evidence review and independent verification without exposing proprietary proof construction internals.
